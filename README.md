# Module-Maker

> Unity 프로젝트에서 특정 모듈(스크립트 / 기능)을 패키지 형태로 관리하기 쉽게 도와주는 도구 / 템플릿  

## 🔎 개요  
Module-Maker는 Unity 프로젝트 내부에서 반복적으로 사용되는 기능이나 스크립트를 **“독립 패키지화(Package-based)”** 하여 관리할 수 있도록 설계된 레포지토리입니다.  
- 패키지 형태로 분리함으로써 재사용성과 유지보수성을 높입니다.  
- Git URL 또는 UPM 기반 관리가 가능하도록 구조화되어 있어, 다른 프로젝트에서도 쉽게 설치/사용이 가능합니다.  

## ✅ 특징 / 장점  
- **독립 패키지 구조** — `package.json`, `Runtime/`, `Editor/` 폴더 기반으로 구성되어 있어 UPM 호환 가능  
- **Git URL 또는 local import 가능** — Unity Package Manager를 통해 손쉽게 추가 가능  
- **재사용 & 공유 용이** — 기능 단위로 패키징되어 있어, 여러 프로젝트 간 공유 용이  
- **유지보수가 쉬움** — 독립 repo로 관리되어 패키지별 버전 관리 및 분리된 개발/배포 흐름 가능
